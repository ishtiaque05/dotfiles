# Setup Nvim
-----------------
1. `./install-nvim.sh`
2. ` mkdir -p ~/.config/nvim `
3. `cd ~/.config/nvim`
4. Copy `**init.vim**` & `**plugins.vim**` in `~/.config/nvim/`
5. `./vim-plug.sh`
